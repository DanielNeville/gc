About the SleepTrack Project

The SleepTrack project consists of a wearable device and associated web application designed to monitor and control night-time bruxism.

The SleepTrack device
  • Wearable device worn on a headband.
  • Electromyography (EMG) sensor to detect muscle activity
  • 9-axis Inertial Measurement Unit onboard.
  • 3.5mm audio connector for biofeedback, and onboard buzzer.
  • LIPO-battery powered with onboard charging circuit.
  • WiFi connectivity.
SleepTrack Web App
  • Control your SleepTrack device from a Desktop Computer, Tablet or Smartphone.
