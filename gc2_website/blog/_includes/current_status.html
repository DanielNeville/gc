Current Status 2016/02/06

Device

  • Breadboard prototype working with IMU and battery shields
  • First PCB rev printed, some issues with footprints, working on rev2
  • Working firmware which collects EMG readings and IMU data, and periodically uploads to GC server

Web App

  • Backend servers (collecting data from device) stable, deployed in a docker instance.
  • InfluxDB and Grafana instances deployed.
  • Angular front-end allows user to interact with device, start realtime mode, or night mode and view nightly stats.
